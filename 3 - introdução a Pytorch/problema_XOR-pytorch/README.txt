	Introdução

o proposito desse mini projeto é realizar os mesmo exercícios anteriores, porem com o uso de uma biblioteca expecializada para facilitar o desenvolvimento e para futuros problemas mais
complexos de forma ráida e eficiente

	Como rodar

para rodar é preciso ter o python instalado e o pytorch, para instalar o pytorch (abra o CMD e coloque pip install torch torchvision torchaudio)

	conclusão

com o uso de uma biblioteca como o pytorch é bem mais rapido de implementar o código, principalmente com redes neurais mais complexas e com multiplas camadas, pois cada um dos neurônios
teria um vetor gradiente diferente, fazer esses calculos manualmente seria muito custoso e demorado, com esse mini projeto foi compreendido o funcionamento básico e sintaxe do pytorch
